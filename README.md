
# Сервис To-Do

Учебный одностраничный проект со списком задач. Приложение позволяет просматривать предустановленные дела, добавлять новые, удалять, дублировать и редактировать задачи. Все изменения сохраняются в `localStorage`, поэтому список не пропадает после перезагрузки страницы.

## Функциональность

- загрузка стартового списка задач при первом открытии;
- сохранение текущего списка дел в `localStorage`;
- добавление новой задачи через форму;
- удаление задачи по кнопке;
- копирование задачи в начало списка;
- редактирование текста задачи прямо в интерфейсе;
- восстановление всех изменений после перезагрузки страницы.

## Стек

- `HTML5`
- `CSS3`
- `JavaScript`
- БЭМ-структура для стилей

## Структура проекта

```text
to-do-pre/
├── blocks/        # стили блоков и элементов
├── fonts/         # локальные шрифты
├── images/        # иконки кнопок
├── index.html     # разметка приложения
├── index.css      # точка входа для стилей
├── index.js       # логика приложения
└── normalize.css  # сброс браузерных стилей
```

## Как запустить

Проект не требует сборки и зависимостей.

1. Склонируйте репозиторий.
2. Откройте файл `index.html` в браузере.

Если хотите запускать проект через локальный сервер, можно использовать любой статический сервер, например расширение Live Server в VS Code.

## Что реализовано по методичке

- создание элементов задач через `template`;
- первичная отрисовка списка из массива задач;
- сбор текущих задач из DOM;
- сохранение и загрузка задач из `localStorage`;
- обработчики для добавления, удаления, копирования и редактирования задач.
